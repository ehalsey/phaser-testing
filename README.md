# Phaser Testing Project

A simple test project using Phaser 3 with TypeScript.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open your browser to `http://localhost:1234` to see the game.

## Project Structure

- `src/main.ts` - Main game file
- `index.html` - HTML entry point
- `tsconfig.json` - TypeScript configuration
- `package.json` - Project dependencies and scripts

## Building

To build for production:
```bash
npm run build
```
